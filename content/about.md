## 안녕하세요, 최민석입니다.

컴퓨터 공학을 전공하고 오픈소스 커뮤니티에서 활동하고 있는 한국인 개발자입니다. 코드를 통해 무언가를 만들고, 그 결과물을 다른 사람들과 공유하는 일에 관심이 많습니다.

### 관심 분야

- **오픈소스 소프트웨어** - 투명하고 누구나 기여할 수 있는 생태계를 믿습니다.
- **프로그래밍 언어** - 시스템부터 웹까지, 다양한 층위의 기술을 다룹니다.
- **자동화와 개발 도구** - 반복적인 일은 기계에게, 창의적인 일은 사람에게.
- **Nix / NixOS** - 재현 가능하고 선언적인 시스템 구성에 관심이 많습니다.

### 기술 스택

주로 사용하는 도구와 기술들입니다:

- **Languages** - TypeScript, Go, Python
- **Tools** - Git, Nix, Neovim, tmux
- **Platform** - Linux (NixOS), GitHub Pages
